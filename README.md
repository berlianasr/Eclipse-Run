# Eclipse-Run
Eclipse Run adalah gim 2D berbasis Python &amp; Pygame yang menghadirkan mekanik unik berupa dua karakter yang dapat dikendalikan bersamaan serta pergantian perspektif yang mengubah arah gravitasi dunia. Pemain mengendalikan Matahari dan Bulan yang harus bekerja sama untuk menyelesaikan rintangan dan mencapai titik akhir setiap level.
